添加微博搜索

添加搜索引擎

点击添加搜索引擎按钮，您可以将“新浪微博”搜索添加到您的浏览器搜索栏中。

如何隐藏微博网页上的搜索框

如果您需要隐藏微博网页上的搜索框，您可以在自定义 CSS 中添加：

.WB_global_nav .gn_search { display: none !important; }

或使用如下 Stylish 样式：

@namespace url(http://www.w3.org/1999/xhtml);
  
@-moz-document domain("weibo.com") {
  .WB_global_nav .gn_search { display: none !important; }
}

或应用如下 AdBlock Plus 规则：

weibo.com##.gn_search

如何去掉网页搜索框中的热搜词

您可以应用如下 AdBlock Plus 规则以屏蔽网页搜索框中的热搜词：

http://s.weibo.com/ajax/jsonp/suggestion?ver=1&cuid=*
