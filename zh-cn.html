安装教程

对不起，脚本当前不支持您使用的浏览器。请考虑选择以下几个支持的浏览器之一。

使用 Mozilla Firefox 浏览器的用户，需要 GreaseMonkey 附加组件以运行本脚本。
如果您已经安装了这个附加组件，点此安装本脚本，在弹出的对话框中确认脚本安装。
使用 Opera 的用户，需要 TamperMonkey Beta 扩展或暴力猴扩展以运行本脚本。
如果您已经安装了以上任一扩展，点此安装本脚本，在打开的页面中确认脚本安装。
注：不支持 Opera 12。
使用 Google Chrome 的用户，需要 TamperMonkey 扩展程序以运行本脚本。
如果您已经安装了这个扩展程序，点此安装本脚本，在打开的页面中确认脚本安装。
使用 Safari 的用户，需要 JavaScript Blocker 扩展或 TamperMonkey 扩展以运行本脚本。
如果您已经安装了以上任一扩展，点此安装本脚本，在打开的页面中确认脚本安装，并根据扩展提示启用脚本。

设置脚本

安装成功后，到新浪微博的页面点击右上角的漏斗图标Y即可进入脚本设置


想要简化界面又不想仔细研究脚本设置？试试导入这个示例脚本设置吧。下载这个文件，在设置窗口中点选脚本选项卡，点击导入打开该文件即可快速设置过滤器。

详细说明

除了在设置界面添加关键字、用户、来源、链接的过滤器外，您还可以使用拖拽或右键菜单快速创建过滤器。

将您要隐藏、折叠或高亮的内容使用鼠标拖动，即可在右上角看到一个黄底的拖放区域，将对应内容拖放到该区域内松开鼠标，根据弹出的对话框选择即可快速创建过滤器。

在微博上点击右键，可以在菜单中找到快速创建过滤器的相关选项。（右键菜单功能仅支持 Mozilla Firefox 浏览器）


如果您之前使用眼不见心不烦脚本，您可以使用设置转换工具将该脚本的设置导入到本脚本中。

关于详细的使用指引，请参见常见问题。

主要功能

  • 根据关键字、作者、来源等隐藏、折叠或高亮微博；
  • 屏蔽推广、粉丝头条、投票等各种微博；防刷屏、版聊；
  • 彻底屏蔽被隐藏的微博以免在手机上再看到他们；
  • 提示查看前自动过滤新微博，新微博桌面提示；
  • 清理版面上的各种部件，去广告；定制微博下按钮顺序；
  • 查看原图，展开 t.cn 短网址，清除发布框默认话题；
  • 自定义网页外观，合并边栏，移除微博空隙，增大正文字号；
  • 仅显示微博列表等基本功能的阅读模式；

脚本安装后大部分功能不会默认启用，您需要到脚本设置中启用对应功能。

隐私权政策

信息收集

脚本不会以任何方式收集您的任何个人信息。但您使用脚本过程中产生的网络访问可能会被您的网络提供商以及对应网站所记录。

网络访问

脚本的网络访问限于访问新浪网的相关接口，限于获取或传输以下数据：

  • 您在设置中选择隐藏、折叠、显示的帐号的信息
  • 如果您选择告知服务器被隐藏的微博，脚本会向微博的服务器发送被隐藏的微博信息

本地存储

脚本会存储您的设置等相关信息到浏览器配置文件中，具体包括：

  • 脚本设置的隐藏、折叠或显示的用户编号、昵称，您的账号编号，及其他设置项；
  • 如果您开启了桌面提示功能，脚本还会记录最近20条显示桌面提示的微博的编号；

任何使用该计算机的用户都可以读写该配置文件，所以建议不要在公共电脑上使用。

其他

脚本不会以您的身份发布任何微博或消息。

如果您使用脚本的导出设置的功能，导出的文件以未加密的方式保存了您对脚本的所有设置，其中可能包括您的相关个人信息，请勿随意传播给他人。
